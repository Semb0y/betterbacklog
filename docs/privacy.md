# Privacy Policy - BetterBacklog

**Last Updated:** February 15, 2026  
**Effective Date:** February 15, 2026

## Introduction

LAARABI Consulting ("we", "our", or "us") operates the BetterBacklog application (the "App") for Atlassian Jira. This Privacy Policy explains how we collect, use, and protect your information when you use our App.

## Information We Collect

### Data We Access

Jira Issue Titles and Jira Issue Descriptions: To analyze the quality of user stories

### Data We Process

- Issue content is sent to **Anthropic (Claude AI)** for analysis
- Analysis results are temporarily stored in **Atlassian Forge Storage** for caching purposes
- No personal user data (names, emails, account IDs) is collected or stored

## How We Use Your Data

We use the collected data solely to:

1. Analyze Jira tickets to provide actionable feedback to improve ticket quality
2. Display analysis results in your Jira interface

## Third-Party Services

### Anthropic (Claude AI)

- Ticket titles and descriptions are sent to Anthropic's API for AI-powered analysis
- Anthropic's privacy policy: https://www.anthropic.com/privacy
- Data is processed in real-time and not retained by Anthropic beyond the API call

### Atlassian Forge

- Analysis results are stored using Atlassian's Forge Storage API
- Data is stored within your Atlassian tenant
- Storage is governed by Atlassian's security and privacy policies

## Data Retention

- **Analysis results**: Cached for 30 days in Forge Storage, then automatically deleted
- **Ticket content**: Not stored by BetterBacklog (only sent to Anthropic API in real-time)
- **User data**: No personal information is stored

## Data Security

We implement industry-standard security measures:

- All API communications use HTTPS encryption
- Anthropic API keys are encrypted and stored securely
- No data is stored outside of Atlassian's infrastructure except during API processing

## Your Rights

You have the right to:

- **Access**: View what data is being processed
- **Delete**: Remove cached analysis results by uninstalling the App
- **Opt-out**: Stop using the App at any time

To exercise these rights, contact us at: **contact@betterbacklog.com**

## Children's Privacy

BetterBacklog is not intended for users under 13 years of age. We do not knowingly collect data from children.
